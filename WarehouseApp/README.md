Задача:
Разработать консольное приложение для учёта товаров на складе с возможностью:
- Добавления, редактирования и удаления записей
- Поиска и фильтрации товаров
- Сортировки по различным критериям
- Расчёта аналитических показателей
- Сохранения и загрузки данных в текстовом файле

Технические требования:
- Язык - C++17
- Компилятор - GCC (MinGW)
- Хранение данных - `std::vector<Product>`
- Формат файла - TXT с разделителем `,`
- Обработка ошибок - Без аварийного завершения
- Платформа - Windows (с поддержкой кириллицы)
